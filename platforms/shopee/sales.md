# Shopee 售中客服话术模板

> 专门针对 Shopee 平台的售中场景设计，涵盖订单确认、物流追踪、支付问题、订单修改等售中服务场景。

## 使用说明

- 话术中的 `[变量]` 需要根据实际情况替换
- 建议按"订单确认、物流话术、支付问题"等场景分组管理

---

## Shopee 售中话术列表（共 10 条）

### 1. 订单确认

**适用场景：** 客户下单后确认订单信息

**话术内容：**
```
您好！感谢您的订单，订单号是[订单号]。订单已确认，我们会尽快为您安排发货。发货后，您可以在Shopee订单页面查看物流信息。预计[时间]内送达，请注意查收。如有问题，随时联系我！
```

---

### 2. 物流追踪

**适用场景：** 客户询问物流状态

**话术内容：**
```
您好！您的订单[订单号]已于[时间]发货，物流单号是[单号]。您可以在Shopee订单页面点击"查看物流"追踪包裹状态。如有疑问，随时联系我！
```

---

### 3. 发货通知

**适用场景：** 主动通知客户订单已发货

**话术内容：**
```
您好！好消息，您的订单[订单号]已发货！物流单号：[单号]。您可以在Shopee订单页面查看物流信息，或直接在[物流公司]官网查询。预计[时间]送达，请注意查收。感谢您的购买！
```

---

### 4. 支付问题处理

**适用场景：** 客户遇到支付失败或支付问题

**话术内容：**
```
您好！关于支付问题，可能的原因有：1）支付方式过期或无效；2）余额不足；3）银行安全验证。请更新您的支付方式，或联系您的银行。更新后，订单会自动处理。如有问题，随时联系我！
```

---

### 5. 订单修改

**适用场景：** 客户要求修改订单信息

**话术内容：**
```
您好！关于订单修改，订单一旦提交，我们无法修改收货地址或商品选择。不过，如果订单还未发货，您可以取消订单并重新下单。取消通常是即时的。需要我帮您处理吗？
```

---

### 6. 地址确认

**适用场景：** 发货前确认客户收货地址

**话术内容：**
```
您好！发货前想确认一下您的收货地址。系统显示的地址是：[地址]。这个地址正确吗？如果需要修改，请尽快在Shopee账户中更新，因为我们即将发货。谢谢！
```

---

### 7. 预计送达时间

**适用场景：** 客户询问送达时间

**话术内容：**
```
您好！根据您的收货地址和选择的物流方式，预计送达时间是[时间]。这是预估时间，实际送达时间可能因物流公司表现而略有差异。发货后您会收到物流追踪信息。如有疑问，随时咨询我！
```

---

### 8. 订单状态查询

**适用场景：** 客户查询订单当前状态

**话术内容：**
```
您好！我为您查询了订单状态。订单[订单号]目前状态是[状态]。[详细说明]。您也可以随时在Shopee账户的"我的订单"中查看最新状态。如有疑问，随时联系我！
```

---

### 9. 发货延迟说明

**适用场景：** 订单发货延迟需要解释

**话术内容：**
```
您好！非常抱歉发货延迟。[延迟原因]。您的订单现在正在准备中，将在[时间]内发货。发货后您会立即收到物流追踪信息。感谢您的耐心等待，再次为延迟道歉！
```

---

### 10. 催单处理

**适用场景：** 客户催促发货

**话术内容：**
```
您好！我理解您等待订单的心情。我查看了订单状态，目前是[状态]。我们正在努力尽快为您发货。发货后您会立即收到物流追踪信息。感谢您的耐心等待！
```

---

## 使用建议

1. **分组管理**：建议在话术管理工具中，将 Shopee 售中话术按"订单确认、物流追踪、支付问题"等场景分组，方便快速选择。

2. **结合工具使用**：配合 [快语](https://www.talkq.vip) 等客服工具，可以统计每条话术的使用频率，持续优化话术库。
