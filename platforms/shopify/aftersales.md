## Shopify 售后话术列表（共 10 条）

### 1. 退货政策说明

**适用场景：** 客户咨询退货流程

```
Hello! Our return policy: 30-day return window from delivery date. Items must be unused with original tags/packaging. Free returns for defective items. Customer pays return shipping for change of mind.

To start a return:
1) Log into your account
2) Go to Orders
3) Select "Return Item"
4) Print return label

Refund processed within 5-7 days after we receive the item. Questions? We're here to help!
```

## 使用技巧

1. **品牌化表达**：用"我们"而不是"系统"，体现人性化和责任感
2. **主动补偿**：提供优惠券、折扣等补偿，化危机为复购机会
3. **透明沟通**：明确说明退款时间、处理流程，减少客户焦虑
4. **主动跟进**：退货确认、物流更新、退款通知都要主动告知
5. **避免推诿**：不要说"这是系统问题"，要主动承担责任

## 独立站售后 vs 平台电商

| 区别 | 独立站 | 平台电商 |
|-----|-------|---------|
| 责任归属 | 品牌自己承担 | 有平台规则约束 |
| 补偿灵活度 | 更灵活个性化 | 相对标准化 |
| 话术风格 | 强调品牌价值 | 强调平台规定 |
| 客户关系 | 注重长期关系 | 单次交易为主 |
