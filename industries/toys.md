# 玩具客服话术

> 专业的玩具行业客服话术模板，涵盖年龄推荐、安全说明、玩法指导、售后处理等全场景。

## 使用说明

- 话术中的 `[变量]` 需要根据实际情况替换
- 玩具行业 80% 咨询集中在年龄适配、安全性、玩法功能三大问题
- 建议配合快捷回复软件使用，提升响应速度

## 售前话术（10条）


## 使用技巧

1. **年龄推荐要准确**：不同年龄适合不同玩具
2. **安全性要强调**：3C认证是家长关注重点
3. **玩法说明要详细**：配合视频更直观
4. **故障处理要及时**：玩具问题影响宝宝体验
